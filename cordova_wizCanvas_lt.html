プロジェクトの作成

Create a project

		
cordova create myApp
    
cd myApp
Cordova ni tsuite wa mo shitteru to omoimasu. Kyou wa, command line kara, atarashi project wo tsukurimasu. Kono project ni "myApp" to namae wo tsukemasu.

WizCanvasプラグインを追加

Add WizCanvas Plugin


		
cordova platform add ios
    
cordova plugin add https://github.com/Wizcorp/phonegap-plugin-wizCanvas

cordova build iOS
WizCanvas wo tsuika shimasu. (sounds like suica) Jikan ga kaka-runode, WizCanvas plugin ni tsuite sesumei shimasu. WizCanvas plugin wo tsukau to, hardware accelerate canvas view wo tsukuru koto ga dekimasu. Sarani, WebGL mo tsukau koto ga dekimasu. Kyo no demo wa, WebGL to Three.js wo tsukaimasu.
