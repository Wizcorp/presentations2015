Plugin Architecture
& Toolin'


  • ☆　Light Weight Core

  • ☆　Cordova

  • ☆　Plugman

    「Plugin Manager」
      • - Cordova 3 introduces a new unified project structure and ships with a very limited API surface - Developers can now compose a version of Cordova with only the APIs they need - Obviously, this means a performance improvement but the real win here is maintenance and upgrading. - plugman, which provides automated discovery, installation, and removal of both core and custom plugins.

New APIs (Continued...)

		// Globalization API Example
    navigator.globalization.getCurrencyPattern(
        'USD',
        function (pattern) {
        	alert('pattern: ' + pattern.pattern + '\n' +
                            'code: ' + pattern.code + '\n' +
                            'fraction: ' + pattern.fraction + '\n' +
                            'rounding: ' + pattern.rounding + '\n' +
                            'decimal: ' + pattern.decimal + '\n' +
                            'grouping: ' + pattern.grouping);
        },
        function () { alert('Error getting pattern\n'); }
    );

All New Command Line Tooling


"The Cordova CLI is a command line tool to build, deploy and manage Cordova based applications."

All New Command Line Tooling


		
		# Get Cordova from npm
		npm install -g cordova
		
		# Create a new project
		cordova create testApp
		
		# Add our target platforms
		cd testApp
		cordova platform add ios android
				
		# Add a plugin
		cordova plugin add https://github.com/Wizcorp/phonegap-plugin-wizSpinner
		
		# Compile our projects
		cordova build
