Create a project

		
cordova create myApp
    
cd myApp
Should now all know what Cordova is! Let me show you how easy it is to use a Cordova plugin.

Add WizCanvas Plugin


		
cordova platform add ios
    
cordova plugin add https://github.com/Wizcorp/phonegap-plugin-wizCanvas

cordova build iOS
Takes a while... Explain what is wizCanvas
